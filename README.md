# GeoThis

A lightweight browser quiz that helps you learn geography by guessing US states or European countries, with multiple-choice and map-click modes.

## Features
- US states and Europe countries
- Multiple-choice and click-on-map modes
- Capitals mode (US + Europe)
- Persistent coloring for correct/incorrect answers

## Run locally
Open index.html in your browser.

## Data sources
- US states: https://cdn.jsdelivr.net/npm/us-atlas@3/states-10m.json
- Countries: https://cdn.jsdelivr.net/npm/world-atlas@2/countries-50m.json
